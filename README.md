# Binary Calculator (Prototype)

This branch contains a minimal Android Jetpack Compose prototype for a Binary Calculator.

What is included:
- Minimal Compose-based MainActivity and UI
- CalculatorEngine (uses BigInteger) with basic operations and bit masking
- ViewModel that connects UI to the engine

To run:
- Open the project in Android Studio and run the `app` module on an emulator or device.

This is an initial prototype. Next steps can include bit toggles, history persistence, word-size selection, better UI, and unit tests.
